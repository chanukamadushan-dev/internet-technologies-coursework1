What is the internet

  • A network of interconnected computers and other devices worldwide. that allows people to exchange information and communicate.

How the Internet works

  • The Internet works using the Client-Server model. The user's computer or mobile phone is called the client, and the computer that stores websites and online data is called the server. When a user types a website address like [www.google.com](http://www.google.com) into a browser, the DNS (Domain Name System) first converts that name into an IP address that the computer understands. The browser then sends a request to the server using HTTP (HyperText Transfer Protocol). The server processes the request and sends the relevant HTML files, images, and other data back to the client. Finally, the browser displays that data and shows the web page to the user.

What are the main services in the Internet

  1. World Wide Web (WWW)

    • Websites & Browsing: Accessing interconnected documents, media, and web applications using browsers like Chrome, Safari, or Edge
    • Search Engines: Services like Google allow users to index and search billions of web pages for information

  1. Communication Services

    • Email: Sending and receiving electronic messages and attachments globally
    • Social Media & Messaging: Platforms that allow real-time text, audio, and media sharing

  1. File Transfer & Storage

    • Cloud Storage & File Sharing: Services like Google Drive, Dropbox, or OneDrive that store data on the cloud

  1. E-commerce & Transactions

    • Online Shopping: Buying and selling physical or digital
    • Digital Banking: Accessing bank accounts, transferring funds, and using mobile payment apps

How the Internet is used in daily life

  • Information Retrieval: Searching for news, weather, and other information online.
  • Communication & Socializing: Staying in touch relies on instant messaging, video calls, and social media platforms
  • Work & Education: The internet makes remote work and digital classrooms possible. It provides seamless access to cloud computing
  • E-Commerce & Errands: From ordering a ride to buying groceries and household goods, applications and websites allow you to compare prices, read reviews
  • Banking & Finance: Managing money is highly digitized. People use secure banking apps and digital wallets to pay bills, transfer funds
  • Entertainment: The way people engage with media has shifted entirely to streaming services (movies, music, podcasts), online gaming, and digital content creation
  • Information & Healthcare: Accessing knowledge is instant. People frequently search for recipe, research medical symptoms using reliable health websites, and consult professionals via telehealth

Role of Web Browsers

  • Web browsers act as the essential bridge between the user and the vast resources of the internet. They function primarily as software agents that retrieve, render, and display digital content from web servers, allowing users to seamlessly interact with websites, web applications, and multimedia

What is DNS and why it is important

  • The Domain Name System (DNS) is the phonebook of the Internet. It translates human-readable domain names (like google.com) into numerical Internet Protocol (IP) addresses (like 192.0.2.1) so web browsers can load Internet resources.It is important simply because computers understand IP addresses, so what the user types in letters and requests is converted into numbers and then retrieved from the server.
